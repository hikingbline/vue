## data与el的2种写法
1.el有2种写法
(1).new Vue时候配置el属性。
//mount挂载的意思  
(2).先创建Vue实例，随后再通过vm.$mount('#root')指定el的值。

## 2.data有2种写法
(1).对象式
data:{name:'张三'}
(2).函数式
data(){
    return {name:'张三'}
}
如何选择：目前哪种写法都可以，以后学习到组件时，data必须使用函数式，否则会报错。

## 3.一个重要的原则：
由Vue管理的函数，一定不要写箭头函数，一旦写了箭头函数，this就不再是Vue实例了。
箭头函数没有自己的this
